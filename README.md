# 🚨 AI-Driven Citizen Grievance Sentiment Analysis System

An advanced NLP and Transformer-based multilingual grievance intelligence platform that automatically analyzes citizen complaints, predicts responsible departments, detects sentiment, assigns complaint priority, and visualizes complaint analytics through an interactive dashboard.

---

# 📌 Project Overview

This project was developed as an AI-powered complaint intelligence system capable of processing civic complaints using:

* Machine Learning
* Deep Learning
* Transformer Models
* Natural Language Processing (NLP)
* Sentiment Analysis
* Multilingual Translation
* Interactive Data Visualization

The system analyzes citizen complaints and predicts:

✅ Concerned Department
✅ Complaint Sentiment
✅ Complaint Priority
✅ Confidence Score
✅ Complaint Analytics

---

# ✨ Key Features

## 🔹 Department Prediction

Automatically predicts the appropriate department for a complaint such as:

* Sanitation Department
* Traffic Department
* Water Department
* Electricity Department
* Police Department
* Housing Department

---

## 🔹 Sentiment Analysis

Classifies complaints into:

* Positive
* Negative
* Neutral

---

## 🔹 Priority Detection

Automatically categorizes complaints into:

* High Priority
* Medium Priority
* Low Priority

---

## 🔹 Transformer-Based NLP

Implemented and compared:

* DistilBERT
* RoBERTa

for advanced complaint classification.

---

## 🔹 Traditional Machine Learning Models

Implemented:

* Logistic Regression
* Random Forest

for performance benchmarking.

---

## 🔹 Multilingual Complaint Support

Supports multilingual complaint analysis using automatic translation.

Languages tested:

* English
* Hindi
* Telugu
* Tamil
* Bengali

---

## 🔹 Interactive Dashboard

Built using Streamlit with:

* Real-time complaint prediction
* Analytics dashboards
* Model comparison visualizations
* Interactive charts
* Live confidence scoring

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Varshit-HariPrasad/AI-Driven-Citizen-Grievance-Sentiment-Analysis-System.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd AI-Driven-Citizen-Grievance-Sentiment-Analysis-System
```

---

## 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

## 4️⃣ Activate Environment

### Windows

```bash
venv\Scripts\activate
```

---

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 6️⃣ Run Streamlit Dashboard

```bash
streamlit run app.py
```
